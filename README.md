SauceDemo Playwright Automation
--------------------------------
A small Playwright automation project where I tested a basic user flow on https://www.saucedemo.com
->> What the test does
-Logs in with valid credentials
-Adds a product to the cart
-Checks if the product name matches in the cart
-Logs out
----------------
How to run it
----------------
git clone https://github.com/AshrafSuzan/saucedemo-playwright.git
cd saucedemo-playwright
npm install
npx playwright install
npm test
--------------
Tech used
-------------
